Accepted BRFSS Data Formats

Upload a .csv or .json file. We auto-detect and validate two formats:

Individual respondent data - Each row is one survey response with BRFSS variable codes.

Key Fields

_STATE State FIPS code (13=GA, 12=FL...)
IYEAR Survey year (2023)
GENHLTH General health (1-5, 7=DK, 9=Ref)
PHYSHLTH Physical health days (0-30, 88=None)
MENTHLTH Mental health days (0-30, 88=None)
DIABETE4 Diabetes status (1-4, 7=DK, 9=Ref)
SMOKE100 Smoked 100 cigarettes (1=Yes, 2=No)
BPHIGH6 High blood pressure (1-4, 7=DK, 9=Ref)

+ 50 more BRFSS variables supported (EXERANY2, CHECKUP1, CVDINFR4, ASTHMA3, etc.)

Example CSV (Raw Survey)

_STATE,IYEAR,GENHLTH,PHYSHLTH,MENTHLTH,DIABETE4,SMOKE100,BPHIGH6,CHECKUP1

13,2023,1,3,0,4,1,1,2
13,2023,4,15,10,3,2,3,1

State 13 = Georgia. Response codes: GENHLTH 1=Excellent to 5=Poor

Summary statistics - Prevalence rates aggregated by state and topic.

Required Fields

year Survey year (1984-2025)
locationabbr State abbrev (CA, TX, NY...)
locationdesc Full state name
topic Health topic (Obesity, Diabetes...)
question Survey question text
data_value Prevalence % (0-100)

Optional Fields

sample_size Number of respondents
confidence_limit_low / confidence_limit_high Confidence interval bounds
break_out / break_out_category Demographic breakdown
